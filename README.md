# Projet PFE : Chatbot Intelligent & Analyse de Feedback

## Contexte
Ce projet vise à développer un assistant intelligent pour les clients Orange, capable de comprendre le contexte de navigation et d'être enrichi par l'analyse automatique des avis clients (Play Store).

## Objectifs
1.  **Collecter** les avis clients de l'application "Orange et Moi Maroc".
2.  **Analyser** ces avis pour détecter les sentiments et les sujets récurrents.
3.  **Développer** un chatbot RAG (Retrieval Augmented Generation) capable de répondre aux questions des clients.
4.  **Intégrer** les insights des feedbacks pour améliorer les réponses du chatbot.

## Structure du Projet
- `src/data_collection/` : Scripts de scraping et de nettoyage.
- `src/analysis/` : Modules d'analyse de sentiments et topic modeling.
- `src/chatbot/` : Moteur du chatbot (RAG, LLM).
- `src/dashboard/` : Interface utilisateur (Streamlit).
- `data/` : Stockage des données brutes et traitées.

## Installation
1.  Cloner le repo (ou télécharger le dossier).
2.  Installer les dépendances :

    pip install -r requirements.txt
